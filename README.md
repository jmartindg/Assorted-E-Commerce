# Assorted E-Commerce

A simple e-commerce website with basic functionalities.

## API Used

- FakeStore API

## Tech Stack

- React
- React Router
- React Use Cart
- React Icons
- TailwindCSS

## Features

- Display all products from the API
- Product dedicated page
- Add to cart
- Cart page (Summary, Update quantity, Remove item, Clear cart)
- Mobile responsive
- Error page

## Deployment

- Netlify
